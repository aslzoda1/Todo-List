Groovy Task Manager — bu JVM platformasining kuchi va Groovy tilining elastikligini namoyish etuvchi interaktiv terminal ilovasi. Ushbu loyiha "Closures" va "Collection API" kabi ilg'or funksional dasturlash elementlarini o'zida mujassam etgan bo'lib, vazifalarni boshqarishni ultra-tezkor va xatosiz amalga oshiradi.

🛠️  𝑻𝒆𝒄𝒉𝒏𝒊𝒄𝒂𝒍 𝑨𝒓𝒔𝒆𝒏𝒂𝒍 (𝑺𝒕𝒂𝒄𝒌)

🔥 Apache Groovy (4.0+): Dinamik va statik tiplash imkoniyatiga ega, Java bilan 100% mos keluvchi yuqori darajali til.

⚙️ JVM Power: Dastur har qanday Java Virtual Machine o'rnatilgan muhitda barqaror va yuqori unumdorlik bilan ishlaydi.

⚡ Functional Logic: Ro'yxatlarni saralash va o'chirishda Groovy-ning kuchli it iteratorlari va closures mantiqidan foydalanilgan.

🛡️ Fail-Safe Engine: Noto'g'ri indeks kiritilganda dasturning favqulodda to'xtashini (Crash) oldini oluvchi xatoliklar nazorati.
